# Product Management API

A simple API for managing products built with **Express**, **Node.js**, **Prisma**, and **PostgreSQL**.

## Features

- Create, retrieve, and update products.
- Built with Prisma for seamless database interaction.

## Technologies

- **Node.js**
- **Express**
- **Prisma**
- **PostgreSQL**
